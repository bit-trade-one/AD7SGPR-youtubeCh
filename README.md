# AD7SGPR YouTube登録者数取得App
このアプリケーションはYouTubeチャンネルの登録者数を定期的に取得し、  
シリアル通信で送信することができます。

### 特徴
YouTube APIを使用して指定したチャンネルの登録者数を取得します。  
取得した登録者数を指定した頻度で更新し、表示します。  
取得した登録者数をシリアル通信で送信することが可能です。
