APIキーの取得方法

1. Google Cloud Consoleにログインします
https://console.cloud.google.com/ にアクセスしてGoogleアカウントでログインします。



2. プロジェクトを選択または作成します
上部のプロジェクトドロップダウンから新しいプロジェクトを作成します。


3. ナビゲーションメニューを開き、「APIとサービス」を選択します
左上のナビゲーションメニュー（三本線アイコン）をクリックして、「APIとサービス」を選択します。
その後、ドロップダウンから「ライブラリ」をクリックします。


4. YouTube Data API v3を探して有効にします
APIライブラリで「YouTube Data API v3」を検索し、そのページに移動します。
ページにある「有効にする」ボタンをクリックします。


5. 認証情報を作成します
APIを有効にした後、左側のメニューから「認証情報」を選択します。
次に「認証情報を作成」をクリックし、「APIキー」を選択します。

6. APIキーを取得します
新しいキーを生成したら、そのキーをコピーして安全な場所に保存します。
このキーがYouTube Data APIの認証情報となります。
※画像下段に書いてある通り制限を設けることをおすすめします。同画面リンクをご確認ください。

本アプリの使用方法

App画像

1. APIキーの入力

YouTube Data API V3から取得したAPIキーを「APIキー:」フィールドに入力します。

2. チャンネルIDの入力

登録者数を知りたいYouTubeチャンネルのURLまたはIDを「チャンネルURLorID:」フィールドに入力します。

3. 取得頻度の設定

「取得頻度（秒）:」フィールドに、何秒ごとに新しい登録者数を取得するかを入力します。デフォルトは10秒です。

4. 登録者数取得開始

すべてのフィールドが適切に入力されたら、「登録者数取得開始」ボタンをクリックします。すると、アプリケーションは指定した頻度でYouTubeチャンネルの登録者数を取得し、 「登録者数:」の横に表示します。

5. シリアル接続

取得した登録者数をシリアルデバイスに送信することも可能です。そのためには、「シリアル接続」ボタンをクリックします。シリアルデバイスとの接続が成功すると、「シリアル接続」ボタンを再度クリックすることで接続を解除することができます。

注意：YouTubeAPIの使用制限により、取得頻度が高すぎると一時的にデータの取得ができなくなる可能性があります。
